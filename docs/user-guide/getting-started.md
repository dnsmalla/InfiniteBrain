# Getting started

1. Install the `.dmg` and open InfiniteBrain.
2. Choose a vault folder. An empty folder will be initialised; an existing
   Obsidian vault will be adopted.
3. Drag PDFs (or `.md`, `.txt`, `.docx`) into the Ingest pane.
4. Watch the log as files flow through extract → atomize → classify →
   reconcile → write.
5. Browse the result in the Vault tab, see relationships in the Graph tab,
   and ask questions in the Query tab.

# Tuning quality

After first launch, the bundled skill files are copied to
`<vault>/.infinitebrain/skills/`. Edit any `SKILL.md` and changes take
effect on the next ingest — no rebuild required. The same applies to
`<vault>/.infinitebrain/rules/*.mdc`.
